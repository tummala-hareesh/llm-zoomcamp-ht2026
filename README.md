## Setup Environment 
```
cd llm-zoomcamp-ht2026/

# Initialize pyproject.toml (only frist time)
uv init --bare

# Add libraries to env. 
uv add [python-libraries] 

# To activate env. 
source .venv/bin/activate

# install ipykernel
pip install ipykernel

# Create kernel based on env.
python -m ipykernel install --user --name="llmzcamp" --display-name="llmzcamp"
```
